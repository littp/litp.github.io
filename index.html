个人建议，刚开始使用坚果云并没有作为主力云盘的话，可以先使用免费版，等待确定使用感受没问题，再做付费选择。

专业版相比较免费版增加的的功能

优点：

1、状态都是实时响应的，并且可以看明白，什么状态什么情况，一目了然，比iCloud强，当然其他网盘也很一目了然

2、可查看历史版本

3、可手动选择某个文件来释放本地空间。

缺点：有点贵

值不值得：鉴于国内网络环境，还是推荐坚果云，iCloud、Google等网络都是硬伤。

iCloud 苹果家的云盘

官网：https://icloud.com

价格：50GB=6元/月，200GB=21元/月，2TB=68/月 （后两者都可家庭共享）

优点：价格绝对是优势，而且apple的隐私性，我个人觉得是非常好的，可以信赖，上传和下载不限制速度，而且可以自由分享，比较适合Apple全套的人来用。

缺点：iCloud的同步机制很惆怅，有些时候文件不会同步上去，也没有手动触发按钮（这一点最坑了，有些时候本地删除了文件，并且也同步成功了，但是总占用空间不变。没有版本记录。

值不值得：iCloud如果试用的是贵州云那么速度还是可以的，只是缺点里提到的莫名的问题很头疼，但我还是开着6元的包月，最起码我的照片是在iCloud上的，起码比某些云盘隐私性高。

Github （被微软收购之后新增了免费的隐私仓库

官网：https://github.com

价格：免费

Github原本是程序员常用的网站，用来存储开源代码，在此我也不太推荐存放个人文件尤其是大文件，这样既浪费空间，又有些占便宜。

优点：

1、版本记录非常详细（基于每次的提交）

2、免费很有诱惑力

3、速度还算可以

4、隐私性高

5、有官方的GUI，相比较命令行来讲很不错了。

6、推送过的文件，即使删除了，也可以从历史记录找回，永久性的（除非删掉仓库）

缺点：

缺点就很明显了，毕竟不是专业云盘来的。

1、需要手动提交推送，这样才会保留历史状态和同步到云端。

2、大于100MB文件无法直接推送，需要安装扩展（参考这里），当然安装完了扩展之后，也一样推送。

3、每次推送的量如果很大，就会很容易失败。

4、本地无法释放空间，本地如果删除了，下次推送的话，云上会对应删除。

5、使用门槛稍微高一丢丢

百度云盘、微云

这两个都感觉是作为分享类、公开化的云盘，搞搞电影啊、安装工具啊就挺好，不适合做私密文件同步，不建议作为自己的个人云盘。

Google drive、OneDrive、Dropbox

这三类均需要科学上网才能使用，鉴于我自己的网络情况，波动很大，所以没有考虑过这些。

DataGrip 如何提取SQL语句

Posted on 2019-03-08

需要解决的问题
在遇到需要修改线上数据的时候，有两种方式生成sql，第一种就是根据提供的需求来手动写SQL，审查，然后在测试环境跑一边确认无误，可以直接扔给运维去执行。
第二种方法就是直接在图形化工具上修改，然后提取修改的SQL，这样提取的只要核对一遍数据的修改是否正确即可扔给运维执行。

很明显第二种可以很方便、很准确的提取出所需要的SQL。

如何解决的简述

DataGrip就是可以按照上述第二种方式来给出所需要SQL的一个工具。

如何解决的详细描述

1 修改需要的数据

首选查询出，需要修改的数据，然后直接在table上修改，双击即可编辑，编辑过后的数据有蓝色背景标识。

2 更改复制模式

默认复制模式，只是复制当前数据，在Database Console 窗口的右侧，可以看到修改复制模式的位置“红色标记”，选择SQL Updates 即可。

3 复制一定要两列起

单独一个字段的复制，系统会默认复制当前字段的值，只有两个字段一起复制，才可以按照“复制模式”来复制出需要的数据，比如这次我修改的是字段“auto_repayment”但复制的是还需要增加随便一列，我选择的是“all_rate”

Google Cloud 释放外网IP

来自Google Cloud 一年免费版的受益者

近几日突然发现上推的时候，访问情况不稳，时断时续，而且出现了访问速度急速下降的问题，就比如说在推特看小视频，以前缓冲几乎是秒级别，现在就会卡顿顿的，看的不爽，自然心情也就不好。

Google Cloud 比 vultr 好一点的地方是服务器的IP可以单独配置，这样就可以既保留我服务器上的资源也可以同时避免被墙的尴尬。

第一步
在自己的 Compute Engine 中修改服务器的外网IP为 “空” 或者 “临时” ，目的是为了释放外网IP做准备。

⬇点击修改按钮，进入修改模式

⬇修改外部IP为“空”或“临时”

⬇记得在底部点击保存按钮，这时外部IP是无占用状态

第二步
释放外网IP

⬇进入VPC网络

⬇点击选中需要释放的IP，点击右上角“释放”

第三步
重新创建IP，回到第一步骤修改的状态，点击外部IP之后有一项是创建外部IP，创建即可。
